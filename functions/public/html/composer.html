noti.ms

noti.ms es un sistema que se basa en links enviados por SMS donde se utiliza la previsualizacion de informacion y visualizacion de contenido en una landing en base al envio masivo por API o excel. El sistema ofrece un link composer que ofrece una facil e intuitiva herramienta para crear la previsualizacion y web app que el usuario va a utilizar y con quien va a interactuar a travez de la plataforma.

Preview y Web Composer
Preview Composer permite crear el template de preview (html to image) que se previsualiza en el cuerpo del SMS a modo de CARD. Web Composer prermite crear un template Web App con el cual el usuario final va a interactual una vez hecho el click en el preview o link.
Datos flexibles
La fleximibilidad del composer permite que los datos sean totalmente flexibles a la hora de crear el preview y web app. De este modo se puede ajustar un excel o una base de datos a traves de API.
Analitics
La plataforma ofrece todo tipo de datos interactivos del usuario, por empezar si previsualizo la imagen del preview hasta toda la interacción completa en la web app. El medio de pago que optó, el modo de comunicación con un operador (whatsapp/llamado, chat) y toda la interacción con el mensaje que luego es reflejado en el panel dentro de cada campaña.
Follow up buenas practicas
Dentro de la web app, el usuario final puede optar desde no recibir mas ese mensaje hasta recibir nuevas notificaciones web o mas SMS.
